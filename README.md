# RideReady WA

A free practice quiz to help Washington State motorcyclists prepare for their DOL endorsement exam.

## What It Does

- Covers all major exam topics: Licensing, Protective Gear, Pre-Ride Inspection, Riding Technique, Cornering, Braking, Street Strategies, Impairments, Group Riding, Passengers & Cargo, and Risk & Responsibility
- Supports both **Two-Wheel** and **Three-Wheel** (trike/sidecar) endorsement tracks
- Randomizes questions each attempt
- Shows live pass/fail status against the 80% threshold as you go
- Displays an explanation after each answer
- Provides a missed-question review at the end

## Tech

Single-file static HTML/CSS/JS — no build step, no dependencies, no backend.
